# claude

Claude Code plugins by and for Phi Labs Ltd.

Right now the marketplace ships one plugin: **`personalities`**, with one output style: **Crypto Bro**.

## Install

Add the marketplace, then install `personalities`:

```
/plugin marketplace add phi-labs-ltd/claude
/plugin install personalities@phi-labs-ltd
```

## Use the Crypto Bro output style

Pick a voice with `/config` and choose `Output style` and select **personalities:crypto-bro**, or jump straight to it:


Go back to default with `/config` and choose `Output style` and select **Default**
